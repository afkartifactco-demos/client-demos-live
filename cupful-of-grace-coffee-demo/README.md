# Cupful of Grace Coffee Demo

Static client preview for a mobile coffee/drink trailer. Built from the public Facebook page cues:

- Business name: Cupful of Grace Coffee
- Tagline: Fueled by faith, grace, and caffeine
- Customer goal: first website with online ordering now and merchandise later
- Menu cues: local favorite coffees, dirty drinks, energy drinks, lemonade, hot chocolate, and baked treats
- Event cue: Summer Makers Market / Dragonfly Village at 20860 OH-251, Fayetteville, OH
- Refined order concept: request-style cart that can later connect to forms, text, Square, Toast, or Messenger

Open `index.html` directly in a browser to review the demo.
