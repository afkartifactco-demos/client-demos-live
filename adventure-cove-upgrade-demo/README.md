# Adventure Cove Upgrade Demo

This is a static website demo for Adventure Cove Miniature Golf in Wilmington, Ohio.

## Preview

Run a simple local server from this folder:

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:4173/
```

## Demo Focus

- Stronger first impression with real course photography
- Clear visit information above the fold
- Mobile-friendly pricing and directions CTAs
- Party pavilion section with a direct phone action
- Photo gallery and embedded map/contact section
