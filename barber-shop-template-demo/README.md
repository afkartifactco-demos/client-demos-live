# Barber Shop Website Template

Reusable static template for independent barbers and small shops.

## What it shows

- Public homepage with services, pricing, location, phone CTA, and appointment booking
- Separate staff login page at `admin.html`
- Private dashboard for client haircut notes, preferred service details, and appointment history
- Local browser storage for demo bookings and notes
- Generic imagery and copy that can be quickly customized for a real shop
