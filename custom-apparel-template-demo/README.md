# Custom Apparel Shop Template

Reusable static template for custom apparel, DTF print, embroidery, vinyl, teamwear, sign, and personalized gift shops.

## What It Shows

- Public homepage with service positioning, product starters, bulk quote flow, gallery framework, and order request form
- Configurable pricing and contact values in `script.js`
- Generic visual system with one blank-shirt hero image and CSS product/gallery mockups
- Copyable order and quote summaries for texting, emailing, or turning into a form submission
- Pickup, shipping, delivery, artwork, deadline, size, color, and quantity fields

## Fast Customization

1. Replace `Custom Apparel Co.` in `script.js`.
2. Replace `hello@example.com` in `script.js`.
3. Update `prices` in `script.js`.
4. Swap `assets/blank-shirt-row.png` with a real client hero image.
5. Replace gallery mockups with client product photos when available.

Open `index.html` in a browser or deploy the folder as a static site.
