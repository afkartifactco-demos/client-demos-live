# Swack & Co. Barber Shop Demo

Static client demo for a barbershop homepage with online booking and a private barber dashboard.

## What it shows

- Public homepage styled around the provided Swack & Co. logo
- Services and pricing cards
- Appointment scheduler that adds bookings to the live schedule
- Separate staff dashboard for haircut notes, preferred barber, and appointment history
- Staff login page at `admin.html`
- Local browser storage so demo updates persist until the browser cache is cleared
