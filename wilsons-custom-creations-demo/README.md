# Wilson's Custom Creations Demo

Static online ordering platform for Wilson's Custom Creations.

## Included

- Product catalog using the supplied customer photos
- Starting prices:
  - Solid brass pendant: $35
  - Custom board games: $30
  - Custom signs: $25
  - Hats: $25
  - Wedding cupcake stand: $75
  - Patches: $12
  - Christmas ornaments: $15
- Product customizer with quantity, personalization, finish, and needed-by date
- Cart drawer with estimated subtotal
- Customer checkout/request form
- Local saved order requests for demo and admin review
- Copyable order summary for texting or emailing Wilson

Open `index.html` directly in a browser.
